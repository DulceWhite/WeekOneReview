How to Push to GitHub

PART ONE

  1. Move into the correct folder using de cd command in Terminal.
  2. If it is the first time pushing to GitHub type the command git init (you will be successfull if you see git:(master) after folder name
  3. type the command git add . to add all files whitin the folder, otherwise you can add specific files by typing the file names after add.
  4. type the command git commit -m"your comment"

PART TWO

  1. Go to GitHub.com
  2. Create a new repository
  3. Name repository and click create Repository (next Green button
  4. Copy the 3 lines of git command

PART THREE

  1. Return to VS code
  2. Copy the three lines into Terminal
  3. Wait for it..
  4. Go back to GitHub
  5. Refresh (Command + R)
  6. Done
